픽셀 - 모니터 화면을 구성하는 가장 작은 단위 
선택한 픽셀에 원하는 색상을 입력해서 그래픽을 나타내는 툴

포토샵은 작업 전 레이어 창에서 레이어가 잘 선택이 되어있는지 확인할 것
포토샵은 작업하는 내내 레이어창을 주시하며 어떤 레이어가 선택되었는지 확인해야한다.
포토샵에서 선택 영역을 지정한다는 것은 일러스트레이터처럼 특정 오브젝트를 선택하는 개념이 아니라 픽셀영역을 지정한다는 뜻이다.

포토샵은 레이어보다 언제나 선택영역이 우선순위가 높다 어떤 작업이든 선택영역이 지정되어 있다면 선택영역 안에서 작업을 우선 처리하고 
선택영역이 지정되어 있지 않다면 레이어 위주로 작업을 처리한다

선택영역은 레이어를 바꾸더라도 그대로 유지가 되기 때문에(레이어와 각각 존재) 작업이 끝난 뒤에는 선택영역을 해제를 해주는 것을 잊지말자 (CTRL+D)

레이어 옮겨주는 단축키 ctrl + [ , ]
레이어 창 빈공간 우클릭 > 섬네일 크기 바꾸는 법
레이어창 섬네일에 좌클릭 > 색이 들어간 픽셀을 선택해주는 기능


색 채우기 EDIT > FILL 
전경색 채워넣기 ALT + DELETE 
배경색 채워넣기 CTRL + DELETE

선택영역을 추가할 때는 shift를 누른채로 드래그 뺄때는 alt키를 누른채로 드래그
ctrl + R 룰러를 활성화 할 수 있고 룰러에서는 가이드라인을 뺄 수 있다


free transform ( ctrl + T)
프리트랜스폼 기능을 사용하여 선택한 레이어 또는 선택한 레이어 안의 선택영역을 대상으로 자유롭게 크기 조절 및 회전을  할 수 있다
이때 shift키를 누르면 고정되어있는 가로세로 비율을 깨고 변형할 수 있으며
ctrl 키를 누른 상태에서 프리 트랜스폼의 변형을 할 수 있는 포인트를 클릭 드래그를 하여 개별적으로 변형 가능하다

캔버스 사이즈 조정
[image >canvas size]
absolute 방식과 relative 방식
absolute 방식은 기본 방식으로 현재 캔버스의 정확한 크기를 수치로 나타낸다 일러스트레이터와 동일하게 계산식을 사용하여 수치를 조정할 수 있다

relative 방식은 상대값을 사용하여 증가하거나 감소할 증감값을 입력하여 기본 캔버스의 사이즈 대비 변경값만 적용하여 수치를 조정할 수 있다

1,alt + 마우스 좌클릭하여 클릭한 지점을 기준으로 픽셀데이터를 복사한다
2,복사한 픽셀정보를 필요한 곳에 드래그 혹은 클릭하여 사용한다(이때 자연스러운 효과를 위해 옵션창에서 브러쉬의 모양은 softround브러쉬를 사용한다)

clone stamp도구는 픽셀을 복사해서 붙여넣기 하는 도구

선택영역 조정

select > modify 메뉴

feather - 선택한 영역의 가장자리에서 뭉개짐 정도를 의미
수치가 커질수록 더 넓게 뭉개지며 선택됨
(픽셀 선택툴을 사용하기 전에 상단 옵션창에서 feather 값을 확인한 후 사용)

expand - 선택한 영역의 가장자리에서 바깥쪽으로 확장하여 픽셀을 더 선택함
contract - 선택한 영역의 가장자리에서 안쪽으로 축소하여 픽셀을 선택함
